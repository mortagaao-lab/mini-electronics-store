# متجر الإلكترونيات المصغّر

## وصف المشروع

مشروع ويب بسيط باستخدام PHP و MySQL لعرض منتجات إلكترونية وطلبها من قبل المستخدمين.

## فكرة المشروع

فكرة المشروع هي إنشاء متجر إلكتروني بسيط.  
المستخدم العادي يستطيع تسجيل الدخول، تصفح المنتجات، وطلب المنتج.  
المسؤول يستطيع إدارة المنتجات، التصنيفات، الطلبات، والمستخدمين من لوحة التحكم.

## طريقة تشغيل المشروع

1. تشغيل XAMPP.
2. تشغيل Apache و MySQL.
3. وضع مجلد المشروع داخل:

C:\xampp\htdocs\

4. فتح phpMyAdmin.
5. إنشاء قاعدة بيانات باسم:

mini_store

6. استيراد ملف قاعدة البيانات:

database/database.sql

7. فتح المشروع من الرابط:

http://localhost/mini_store/
